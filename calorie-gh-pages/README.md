# 轻食的「热量谎言」—— H5 融媒体新闻

上海市消保委60款轻食测评数据可视化，支持微信分享。

## 一键部署到 GitHub Pages

1. 在 GitHub 新建仓库，把本文件夹推送上去
2. Settings → Pages → Source 选 **Deploy from a branch**，分支选 `main`，保存
3. 等一分钟，访问 `https://你的用户名.github.io/仓库名/`

## 微信分享封面图

替换 `index.html` 中的 og:image 链接：

```html
<meta property="og:image" content="https://你的用户名.github.io/仓库名/share-cover.jpg">
```

并制作一张 600×600 的 JPG 封面图放到同目录下。

## 数据来源

上海市消费者权益保护委员会《60款轻食样品调查结果》2025年1月3日发布
